Homework 4 - Arithmetic Package

To further explore the mechanics of computer arithemetic, implement four different classes: simple adder, adder with carry look-ahead, simple mutilply, and booth's algorithm.

Each should simulate the bit-level operations of these diffferent functional units. For example, the simple adder should loop through the bits of the two values, and determine the sum of each bit (with carry).

Two utility classes, BitString32 and BitString64 have been written to aid in the development of this task. Additional three shells of code have beenn provided. The shell for multiply has been intentionally omitted. Use of these classes is optional.

Task:

Choose one langauge (C, C++, Java, LISP, PERL, MIPS Assembly) and implement a simulator for the four arithemtic algorithms described in the text book:

  • simple adder
  • adder with carry look-ahead
  • simple multiply
  • booth's algorithm for mulitplication

You do not need to simulate the actions graphically. Just simulate them mechanically.
